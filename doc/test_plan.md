# VSync テスト計画書

## 1. 概要

### 1.1 プロジェクト概要
本テスト計画は、FPGA (Xilinx Spartan UltraScale+) 向けハードウェアPOSIX対応RTOS内蔵RISC-V RV32IMプロセッサ「VSync」の検証戦略を定義する。

### 1.2 テスト目標
- RISC-V RV32IM ISA仕様への完全準拠
- 5段パイプラインの正確な動作
- ハードウェアRTOS機能の正確性
- POSIX互換レイヤーの動作保証
- バス・メモリサブシステムの信頼性
- ペリフェラルの機能正確性
- FPGA合成可能性とタイミング制約充足

### 1.3 テスト方法論
- **アサーションベース検証 (ABV)**: SystemVerilog Assertions (SVA) を活用
- **タスクベース検証**: 再利用可能なテストタスクによるモジュラー検証
- **カバレッジ駆動検証**: 機能カバレッジとコードカバレッジの両方を測定
- **制約付きランダム検証**: 制約ランダムテストによる境界値探索

### 1.4 テストベンチディレクトリ構造
```
tb/
├── common/           共通ヘルパー・BFM
│   ├── clk_rst_gen.sv        クロック・リセット生成
```
